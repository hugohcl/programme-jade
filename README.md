# Programme de Jade

Programme d'entrainement Keep Cool pour Jade : 2 seances guidees pas a pas (machines Technogym expliquees), un bonus cardio facultatif, un minuteur de repos et un compteur de seances.

Page statique unique (`index.html`), sans dependance externe : la police est inline, les donnees (series cochees, poids, compteur) restent dans le localStorage du navigateur.

Publie via GitHub Pages.
